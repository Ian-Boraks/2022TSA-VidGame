In this game, you are a historian that stumbled his way into a broken time portal. You are sent back in time and need to collect the tokens to restore the time line. Because of the incident, some physics don't work as expected. Waterfalls flow upwards and you might be teleported around at random. Because of the randomness of this physics change, you might get stuck outside of the playable area, if this happens just press R on your keyboard.

CONTROLS

A, S, D, SPACE

A: Moves the player to the left
D: Moves the player to the right
S: Makes the player crouch (reduces hitbox by 1/2)
SPACE: Makes the player jump
R: Makes the player respawn at the beginning of the level.

DIFFERENT OBJECTS

Lava: If touched, it will force you to respawn and you will lose some time/score

Waterfall: If touched, the player will accelerate upwards.

Background: This object has no collision on it and can be walked through

Tokens: If touched, they will disapear and the player will gain time/score

Stair: The user will automatically walk up them

Ground: Is a solid object with collision

HOW TO WALL JUMP

To wall jump, the player must jump towards a wall and while touching the wall, the player must press the opposite directional key while holding space.

Example 1 (Basic Wall Jump):

Hold SPACE and A

At apex of jump, continue holding SPACE, release A and press D

Example 2 (Chained Wall Jump)

Hold SPACE and D

At apex of jump, continue holding SPACE, release D and press A

Once colliding with the next wall, continue holding SPACE, release A and press D

Repeat to climb up the wall

EDITOR

To use the editor, hit the enter key while focused on the game window. (To exit the editor, hit the enter key a second time)

It should look somthing like this:

To change which type of object your are making, use the number keys on your keyboard.

The number and corresponding object type is listed below:

  1. Ladder
  2. Trap
  3. Token
  4. Solid Ground
  5. Stairs
  6. Background Element

Once you have your object type selected, click and drag on the game canvas to create the object. If you want to change the snapping of the editor use [ and ] to either decresee or increase the grid size by 5. The current size of the snapping grid is shown at the end of the second line of the editor text box. (You can also undo the last placed object by using ctrl + z)
